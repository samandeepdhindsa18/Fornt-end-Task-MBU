Front-End:

1. Application can be login using username available at https://reqres.in/api/login. 
   [Possible username could be: cerulean, true red, tigerlily, blue turquoise, aqua sky].
   Password can be any combination.

2. On login, User can view posts from https://jsonplaceholder.typicode.com/posts and can remove the post.

3. User can view the reports which are displayed in same color in component as given in data corresponding to report.

4. User can logout and cannot see the data.

Application can run locally by download the code and run the command in server directory: "npm run dev"
or also runs in production at link "https://radiant-dawn-47485.herokuapp.com/"
